# Agenda Fatec

Aplicação web que gerencia o agendamento de salas de aula da FATEC Jahu-SP

## Introdução

Nosso projeto teve início através do Projeto interdisciplinar, nosso propósito é desenvolver um software que seja responsável por fazer o gerenciamento das salas, com o objetivo de facilitar a questão do agendamento e utilização das salas da Fatec Jahu.
